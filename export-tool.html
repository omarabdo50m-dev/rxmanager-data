💊 RxManager → GitHub Export Tool

Reads your IndexedDB data and downloads it as properly formatted JSON files ready to upload to your GitHub repo.

📋 Instructions

1
Open this file in the same browser where you use RxManager Pro (so it can read your IndexedDB)
2
Click Read My Data below to load your drugs and products
3
Download drugs.json and products.json
4
Go to your GitHub repo → drugs/ folder → upload both files (they will replace the empty ones)
5
Commit with message: feat: migrate existing drug library from IndexedDB
